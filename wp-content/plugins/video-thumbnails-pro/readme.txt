== Description ==

Enhance the power of Video Thumbnails with more powerful settings

== Installation ==

1. Install and activate the free 'Video Thumbnails' plugin through the 'Plugins' menu in WordPress
1. Upload the `/video-thumbnails-pro/` directory to the `/wp-content/plugins/` directory
1. Activate the 'Video Thumbnails Pro' plugin through the 'Plugins' menu in WordPress
1. Visit Settings > Video Thumbnails Pro in the WordPress dashboard and enter your license key to enable plugin updates as well as configure any additional settings

== Frequently Asked Questions ==

= Why isn't it working? =

Video Thumbnails Pro is an extension to the free version of Video Thumbnails, so you'll need to make sure you're running the latest version of both plugins.
